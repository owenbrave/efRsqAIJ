# 项目简介

## 核心功能
- 用户管理
- 店主管理
- 商品信息管理
- 商品类型管理
- 交流论坛
- 系统管理
- 订单管理
- 个人中心
- 支付宝沙箱支付

## 适用场景
- 超市企业
- 商品零售业
- 线上线下结合的零售模式

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
